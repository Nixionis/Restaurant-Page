### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Nixionis/Restaurant-Page.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build project with Webpack (if js code was changed)
   ```sh
   npm run build
   ```
